## Summary

Opinionated installation-and-workflow guide for using Git and GitHub with R, RStudio, and R Markdown. Covers installation, connecting Git to GitHub and RStudio, foundational workflows (clone, commit, push, pull, branch, merge), and the unique usage patterns that arise in data science vs. pure software development contexts.

## Key Points

- Targets R data analysts and package developers who need to version-control their work, not just pure software engineers
- Structure: Installation → Connect Git/GitHub/RStudio → Early GitHub Wins → Basic Git Concepts → Daily Workflows
- Emphasizes that data science git use has a "different vibe" from software development — focuses on key daily workflows rather than exhaustive git theory
- Resource for the pharma Git workshop context (see [[Version Control with Git & RStudio Workshop]])
