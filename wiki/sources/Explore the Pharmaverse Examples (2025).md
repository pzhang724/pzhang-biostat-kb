## Summary

Describes the Pharmaverse Examples site (pharmaverse.github.io/examples/) — a curated, community-maintained collection of end-to-end worked examples covering the full clinical reporting pipeline. The site spans SDTM construction, ADaM derivation, TLG production, interactive teal apps, automated reporting, and eSubmission preparation.

## Key Points

- Site organized by pipeline stage: SDTM (DM, VS, AE), ADaM (ADSL, ADTTE, ADVS), TLG outputs, interactive apps, automated reporting (slides/docs), eSubmission
- Available via step-by-step reading or via interactive Posit Cloud environment (no local setup needed)
- Community-driven: contributions of new examples and improvements welcomed via GitHub
- Mission: help teams understand how packages fit together, share cross-team workflows, and build reproducible end-to-end pipelines
