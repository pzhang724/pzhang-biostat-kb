# Outstanding User Interfaces with Shiny

**Raw file**: `raw/sources/Outstanding User Interfaces with Shiny.md`  
**Author(s)**: David Granjon, PhD  
**Year**: 2021 (ongoing updates)  
**Type**: textbook-chapter  
**Venue**: unleash-shiny.rinterface.com

## Summary

Advanced reference book filling the gap between beginner Shiny (Mastering Shiny by Hadley Wickham) and production-grade deeply customized Shiny applications. Covers HTML/CSS/JavaScript internals of Shiny, custom UI frameworks, package development for Shiny, and production deployment patterns.

## Key Points

- Targets developers who want Shiny apps that look and behave like native web applications
- Complements Wickham's *Mastering Shiny* (which covers best practices for correct, readable Shiny) with deep customization
- David Granjon is prolific R package creator, educator, and community contributor in R/Shiny
