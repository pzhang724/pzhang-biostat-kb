## Summary

The full method repository table from the CAMIS website. Contains 66+ statistical methods with R, SAS, and Python implementation articles and cross-language comparison pages organized by category. This is the primary deliverable of the [[CAMIS Working Group]]. Each row provides per-language implementation articles and an explicit comparison article documenting discrepancies.

## Key Points

- **Coverage** (66+ methods across 10 categories): summary stats (rounding, skewness/kurtosis), GLMs (t-tests, ANOVA, ANCOVA, MANOVA, linear/logistic regression), non-parametric (Wilcoxon, Kruskal-Wallis, Friedman, Hodges-Lehmann), categorical (binomial, McNemar, CMH, proportions CIs), correlated data (**LMM, MMRM, GLMM, GEE**), MI (linear regression, PMM, tipping point, reference-based/joint), survival (KM/log-rank/Cox, AFT, weighted log-rank, RMST, CIF, recurrent events, tobit), **sample size** (superiority, NI, equivalence, BioEq, group sequential), **causal/ML** (propensity score matching/weighting, PCA, XGBoost), other (survey statistics)
- MMRM and GSD are now covered (important for regulatory submissions)
- Bayesian and some group sequential binary endpoint sections still in development
- FDA Statistical Software Clarifying Statement motivation: language choice is not prescribed; reliability must be demonstrated
