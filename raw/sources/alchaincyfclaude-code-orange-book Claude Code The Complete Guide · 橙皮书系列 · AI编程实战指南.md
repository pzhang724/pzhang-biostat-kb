## Who Is This For

- **Engineers** who want 10x productivity — let Claude Code handle boilerplate, debugging, testing, and CI/CD while you focus on architecture and product thinking
- **Product Managers** who want to build MVPs themselves — turn a weekend into a working prototype without waiting for dev resources
- **Entrepreneurs** building one-person companies — Claude Code gives one person the development capacity of a small team

No prior AI programming experience required. We start from zero but don't linger on basics.
